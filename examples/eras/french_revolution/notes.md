# French Revolution (1789–1799)

This era is provided as a starter example for the demo environment. Teachers can add their own eras
by creating a new folder under `examples/eras/` with an `events.json` and supplementary notes.

The `events.json` file contains chronological events with signal impacts that the runner uses to
build a turn-based artifact.

---

Feel free to modify these notes or replace them with class-specific commentary.
