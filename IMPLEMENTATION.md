# esm-spec v0.1 — Reference Implementation Notes

This repository is a reference implementation of **esm-spec v0.1**.

Implements:
- Signal layer (teacher-editable)
- Deterministic feature aggregation
- Policy evaluation that produces a decision + rationale
- Turn artifact output (inspectable)

Non-goals (for this demo):
- Full governance suite
- Identity/auth layers
- Production reliability guarantees
